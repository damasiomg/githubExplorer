# GitHub Explorer

## What's it?
This is a project to list public details about GitHub repositories. This was built with TypeScript, React and Webpack (without creat-react-app).

## Sample
![](githubExplorer.gif)

## Requirements:
1. `node 10.21.0 or >`

## Running the project:

1. `cd githubExplorer`
2. `yarn`
3. `yarn dev`
